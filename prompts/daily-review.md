# Prompt: daily-review

Run an end-of-day review.

## Output
- Accomplished
- Progress by Project/Area
- Insights
- Blocked
- Questions
- Tomorrow focus (top 3)
- Open loops

## Rules
- Keep it brief.
- Use completed tasks and updated items from today.
- Write a daily note in Inbox for the day and update its Status.

## MCP
Query Tasks completed today and Tasks edited today.
Create or update the daily Inbox note.
