# Prompt: inbox-processor

Process Backlog tasks and Resource inbox.

## Output
- Items to update and link
- Items to convert to Projects or Resources
- Items needing more context

## Rules
- Keep items in Backlog if unclear.
- Use PARA links.
- Ask one short question if needed.

## MCP
Query Tasks with Status = Backlog.
Query Resources with Status in (Inbox, To Review).
Update or create links.
