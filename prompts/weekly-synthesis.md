# Prompt: weekly-synthesis

Create a weekly synthesis.

## Output
- Week at a glance
- Key themes
- Major insights
- Progress by Project
- Questions emerged
- Energy audit
- Connections made
- Next week intentions
- To process

## Rules
- Keep it short.
- Use Tasks, Projects, and Resources updated this week.
- Write a weekly Resource note.

## MCP
Query Tasks, Projects, and Resources edited this week.
Create a Resource note.
